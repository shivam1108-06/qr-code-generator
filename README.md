# QR Code Generator 📱

A modern QR Code Generator built using Flask, Python, Bootstrap 5, and QRCode Library.

## Features

* Generate QR Codes instantly
* Support for Text and URLs
* Download generated QR Codes
* Recent QR History
* Responsive UI
* Loading Animation
* Real-time Date & Time
* Modern Bootstrap Design

## Technologies Used

* Python
* Flask
* QRCode Library
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Installation

Clone Repository

```bash
git clone https://github.com/shivam1108-06/qr-code-generator.git
```

Move to Project Folder

```bash
cd qr-code-generator
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Project

```bash
python app.py
```

Open Browser

```text
http://127.0.0.1:5000
```
